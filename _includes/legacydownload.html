Ricochet Refresh (Legacy)

On this page you can download the old (legacy) version of Ricochet Refresh. This version is no longer supported and should only be downloaded by those trying to communicate with a user who has not yet upgraded to Ricochet Refresh V3. This version of Ricochet Refresh will stop functioning once tor's support for v2 onion services ends in October 2021. You must upgrade to Ricochet Refresh V3 before that date to be able to continue to communicate.

Legacy Ricochet Refresh has a pre-release for Linux, Mac OS X, and Windows. We have included tor 0.3.5.8 in the release.

All releases are GPG signed by fB89 8DF4 10CF 4EA2 E6B3 8A7D 260C 7A7E AE9D 91A0

Linux

Download 1.1.4e for Linux

Installation Instructions

  • Extract the archive, and run the Ricochet Refresh binary.

Windows

Download 1.1.4e for Windows

Installation Instructions

  • Check GPG Signature: Signature
  • Download and run the installer.

Mac OS X

Download 1.1.4e for MacOS

Installation Instructions

  • Check GPG Signature: Signature
  • Click the downloaded .dmg file when it appears.
  • Drag the icon Ricochet Refresh into your Applications folder.
